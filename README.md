# PC Usage Tracker

PC Usage Tracker is a Windows application designed to help users monitor and manage their device usage.  
It tracks active time spent on applications â€” all processed locally and never transmitted externally.

## Features

- **Real-time usage tracking**  
  Monitor how long your PC has been active and which applications you've spent time on.

- **Local data processing**  
  All data stays on your device. Nothing is uploaded or shared.

- **Background operation**  
  The application can run silently in the background to provide continuous tracking.

- **Clean, modern UI**  
  Built with WinUI for a smooth and responsive experience.

---

## Privacy

PC Usage Tracker does **not** collect, store, or transmit personal data.  
All usage information stays on your device.  
For full details, see the [Privacy Policy](https://enoseven7.github.io/UsageTimerWinUI/).

## Installation

PC Usage Tracker will be available through the Microsoft Store (once approved).  
Alternatively, you can clone the repository and build the project using Visual Studio 2022 or later.

## Requirements

- Windows 10 version 1809 (build 17763) or later  
- .NET 8 Desktop Runtime  
- x64, x86, or ARM64 device
